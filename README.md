# Analizador de Textos en Python 📊

Este proyecto es una herramienta interactiva desarrollada en Python utilizando Programación Orientada a Objetos (OOP). Su propósito es analizar textos para calcular métricas como el total de palabras, palabras únicas, longitud promedio, y la frecuencia de uso de cada término utilizando diccionarios y sets.

## 🛠️ Requisitos

* **Python 3.12 o superior.**
* No se requieren librerías externas ni instalaciones adicionales (el código utiliza únicamente funciones integradas de Python).

## 🚀 Cómo ejecutar el programa

1. Descarga o clona este repositorio en tu computadora.
2. Abre tu terminal o consola de comandos.
3. Navega hasta la carpeta donde guardaste el archivo.
4. Ejecuta el siguiente comando:

   ```bash
   python nombre_de_tu_archivo.py
